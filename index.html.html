Niqat Al-Daef

A Lightweight Web Vulnerability Scanner built with Python

About the Project

Niqat Al-Daef is an educational web vulnerability scanner designed to help students and beginner developers identify and understand common security flaws such as SQL Injection, Cross-Site Scripting (XSS), and Insecure HTTP Headers. Developed using Python, the tool aims to make cybersecurity more accessible, ethical, and practical for learning environments.

Key Features

  • Simple and user-friendly command-line interface (CLI)
  • Modular design ‚Äì each vulnerability has its own scanner
  • Educational explanations for each finding
  • Offline and ethical usage only (safe test environments)
  • Beginner-friendly output with suggested fixes
