Niqat Al-Daef - Web Application Vulnerability Scanner
==================================================
A lightweight, Python-based scanner designed for beginners to detect common vulnerabilities like SQLi, XSS, and missing HTTP headers.

Usage:
1. Run `pip install -r requirements.txt`
2. Execute `python main.py`
3. Enter your target URL (use DVWA, bWAPP, or Juice Shop).
4. Select scan modules and review the results.

Disclaimer:
This tool is strictly for educational use only.
